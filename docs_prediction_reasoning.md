# 基金涨幅预测推理依据（V1 -> V1.5）

> 目的：为“投资辅助”场景提供可解释的预测依据，不做收益承诺，不构成投资建议。

## 1. 预测目标定义

- 短期（1-7 个交易日）
  - `up_probability`：未来窗口累计收益 > 0 的概率
  - `expected_return_pct`：未来窗口期望收益率
  - `confidence`：本次预测可信度（由数据完整性、市场状态稳定性、因子一致性共同决定）
- 中期（1-3 个月）
  - 同上，但更强调风格、估值与宏观变量
- 风险补充（建议新增）
  - `max_drawdown_risk`：未来窗口最大回撤超阈值概率

## 2. 推理链路（从市场到基金）

1. 市场状态判断（Risk-on / Risk-off / 震荡）
2. 风格与行业方向判断（成长/价值、大盘/小盘、行业强弱）
3. 基金暴露映射（该基金更接近哪类风格与行业）
4. 价格与波动结构（动量、反转、波动、回撤）
5. 资金与流动性（成交活跃度、资金净流入/流出）
6. 形成收益预测 + 风险校正 + 置信度

## 3. 因子体系（按优先级）

### 3.1 市场行情因子（必须有）

- 宽基指数收益与动量：沪深300/中证500/创业板（1D/5D/20D）
- 市场波动率：指数历史波动、振幅
- 市场广度：上涨家数占比、涨停/跌停比（可选）

作用：
- 判断市场是否支持“普涨”或“防御”风格
- 作为全局先验，修正单只基金信号

### 3.2 风格与行业因子（中高优先）

- 风格相对强弱：成长 vs 价值，大盘 vs 小盘
- 行业相对强弱：近 5/20 日行业排名
- 行业轮动速度：强势行业是否持续

作用：
- 将“市场涨跌”转化为“该基金是否受益”

### 3.3 基金自身行为因子（必须有）

- 净值动量：近 1/5/20 日收益
- 波动与回撤：20 日波动率、最近回撤幅度
- Beta/相关性：对核心指数敏感度
- 跟踪偏离（指数基金）：跟踪误差、折溢价（如可取）

作用：
- 度量基金在当前环境下的“弹性”与“脆弱性”

### 3.4 宏观与利率因子（中期重点）

- 无风险利率（中债收益率）
- 流动性代理（Shibor/DR007）
- 海外扰动（美元指数、美债收益率，若可得）

作用：
- 对中期方向影响更大，对短期作为风险修正

### 3.5 资金面因子（可选增强）

- 北向资金净流入/流出
- ETF 资金流向（宽基/行业）

作用：
- 提升短期时效性，改善拐点识别

## 4. 分 horizon 推理逻辑

## 4.1 短期（1-7 天）

主要依赖：
- 市场短动量 + 基金短动量 + 波动收缩/放大 + 资金面快变量

典型逻辑：
- 市场 risk-on 且基金与强势风格同向时，提高 `up_probability`
- 若波动突然放大且回撤扩大，下调 `confidence` 与 `expected_return_pct`

## 4.2 中期（1-3 月）

主要依赖：
- 风格持续性 + 估值分位 + 宏观流动性 + 基金中期趋势

典型逻辑：
- 风格延续 + 估值不过热 + 利率环境友好 -> 上调中期预期
- 估值偏高 + 流动性收紧 -> 预期与置信度同步下调

## 5. 可解释输出建议（用于 App“预测依据”）

建议后端统一返回结构（示例）：

```json
{
  "code": "110022",
  "horizon": "short",
  "signal_summary": "市场偏风险偏好，基金动量与风格同向",
  "confidence_interval_pct": [-0.8, 2.1],
  "top_factors": [
    {"name": "近5日动量", "value": 1.23, "contribution": 0.31},
    {"name": "20日波动率", "value": 1.87, "contribution": -0.22},
    {"name": "行业强弱", "value": 0.64, "contribution": 0.16}

  "confidence_breakdown": {
    "data_quality": 0.90,
    "regime_stability": 0.72,
```

前端展示建议：
- 一句话结论（signal summary）
- 置信区间
- 正负贡献 Top 因子
- 风险标签（红/黄/绿）

## 6. 你项目当前状态 vs 建议升级

- 当前已实现：
  - 净值、日涨跌、20 日波动
  - 规则基线推理与基础解释
- 主要缺口：
  - 缺“市场状态层”（宽基、风格、行业）
  - 缺“资金面与宏观层”
  - 解释字段仍偏静态，未绑定真实因子值

建议按成本优先级补齐：
1. 接宽基指数行情（先日频，低成本高收益）
2. 接风格/行业相对强弱（至少 5D/20D）
3. 把 explain 从“固定文案”改为“实时因子贡献”
4. 增加 `max_drawdown_risk` 与风险标签

## 7. 评估指标（上线前必须看）

- 离线：
  - 准确率、AUC、F1（方向）
  - 回测年化、最大回撤、夏普（策略层）
- 在线：
  - 覆盖率（有预测基金数 / 基金池）
  - 解释可用率（有因子贡献的请求占比）
  - 接口延迟与失败率
